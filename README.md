## Local Development

1. Clone the repository
2. Open `index.html` in a browser or run a local server:
   ```bash
   python3 -m http.server 8000
   ```
3. Visit `http://localhost:8000`

## Deployment

This is a static site with no build process required. Simply upload all files to any static hosting service:

- Cloudflare Pages
- Netlify
- Vercel
- GitHub Pages

## Privacy

All password generation happens locally in your browser. No data is transmitted to any server.
